# Latest release

The latest release is **v0.1.0** (2026-06-25).

A local, deterministic toolkit that turns meeting text into structured
organizational memory — no network calls, no external models or embeddings.

- Full notes: [v0.1.0](v0.1.0.md)
- Changelog: [CHANGELOG.md](../../CHANGELOG.md)
- Release checklist: [release_checklist.md](../release_checklist.md)

## Highlights

- Complete pipeline: ingestion, extraction, storage, recall, analytics, reports.
- A ten-subcommand `organizational-memory` CLI plus reproducible demos.
- Hardening (logging, validation, benchmarks) and a full documentation set.

## Limitations

Local-only and lexical: no server, authentication, semantic search, LLMs, or
embeddings. See [known limitations](../../release_artifacts/known_limitations.md).
